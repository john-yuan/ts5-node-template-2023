# README

Dependencies:

- TypeScript 5
- ESLint
- Prettier
- Lint-staged
- Husky

## Get started

```bash
# install dependencies
pnpm i

# build and run
pnpm dev
```

## Commands

```bash
# build to js
pnpm build

# build and run
pnpm dev

# format code
pnpm format

# lint code
pnpm lint

# lint and fix
pnpm lint:fix

# lint staged
pnpm lint-staged
```
